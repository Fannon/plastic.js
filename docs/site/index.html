What is it about?

plastic.js is a JavaScript framework that simplifies the common tasks of data retrieval, processind and representation. It is not ment to replace existing visualization / data processing libraries but to integrate them into one bigger, coherent framework.

To learn more, please read into the documentation!

Simple Example

plastic.js is simple to use! No programming knowledge is requred, just write a HTML Embed Code to provide options and the data:

Embed Code

<div id="table-ask-query" class="plastic-js" style="height: 280px; width: 100%;">

        <script class="plastic-data" type="application/json" data-url="data/countries.json" data-format="sparql-json"></script>

        <script class="plastic-display" data-display-module="discrete-bar-chart" type="application/json">
            {
                "staggerLabels": false,
                "tooltips": true
            }
        </script>

    </div>
