staticplastic.processElement

Process a specific plastic element

This initiates all the main steps necessary to create a Data Display

The Main Steps are: 1 IF Query provided Calling the Query Parser 2 IF Data URL provided Getting the Data via AJAX 2 IF Schema provided Calling the Schema Parser 4 ALWAYS Calling the Data Parser 5 ALWAYS Calling the Display Module
