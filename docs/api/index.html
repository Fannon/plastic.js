plastic.js

plastic.js is a JavaScript library that lets you make your data visible! This project is currently in development and not production ready!

Getting Started

Download either the whole repository (files are in the dist/ folder) or directly here: Production JavaScript (minified) and CSS (minified Development JavaScript and CSS

In your web page:

<script src="jquery.js"></script>
<script src="d3.min.js"></script>
<script src="dist/plastic.min.js"></script>
