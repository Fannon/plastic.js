Contributing to plastic.js
^^^^^^^^^^^^^^^^^^^^^^^^^^
There are several ways to contribute to plastic.js. No need to be a programmer in some cases!

* If you have suggestions or have found a bug, please report it to the `issue queue <https://github.com/Fannon/plastic.js/issues/>`_
* Developed a new module, improvement or feature? Fork this `repo <https://github.com/Fannon/plastic.js/>`_ and send a pull request!

Extending plastic.js
^^^^^^^^^^^^^^^^^^^^
plastic.js is built to be easily extensible. In this section of the documentation you will find some
