This example snippet contains all available options with their default values:

<script class="plastic-display" data-display-module="discrete-bar-chart" type="application/json"> 
{
    "tooltips": true,
    "showValues": true,
    "transitionDuration": 350
}
</script>

Properties in detail:

Label Staggering

KeywordstaggerLabels
DescriptionToo many bars and not enough room? Try staggering labels.
Typeboolean
{
    "staggerLabels": false
}

Tooltips

Keywordtooltips
DescriptionShow tooltips
Typeboolean
Defaulttrue
{
    "tooltips": true
}

Display Values

KeywordshowValues
DescriptionShow the bar value right on top of each bar.
Typeboolean
Defaulttrue
{
    "showValues": true
}

Transition Duration

KeywordtransitionDuration
DescriptionDuration of the animation in milliseconds.
Typenumber
Default350
{
    "transitionDuration": 350
