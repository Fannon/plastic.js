Pie Chart
=========

Short Description
^^^^^^^^^^^^^^^^^
This module features a simple Pie Chart diagram. It uses some animations and provides some options.

Dependencies
^^^^^^^^^^^^
The Pie Chart utilizes the NVD3 library (http://nvd3.org/) for its appearance and animations.
